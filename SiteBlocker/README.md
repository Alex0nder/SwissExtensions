# Site Blocker

Минималистичный блокировщик отвлекающих сайтов для Chrome.

## Возможности

- **Вкл/выкл** — один клик включает или отключает блокировку
- **Список доменов** — добавляйте и удаляйте сайты
- **Фокус** — блокирует main_frame и sub_frame (страницы и iframe)

## Установка

1. Откройте `chrome://extensions`
2. Включите «Режим разработчика»
3. «Загрузить распакованное расширение» → выберите папку `SiteBlocker`

## Использование

1. Нажмите на иконку расширения
2. Включите блокировку переключателем
3. Добавьте домены (например: `twitter.com`, `facebook.com`, `youtube.com`)
4. Сайты из списка будут блокироваться при попытке открыть
