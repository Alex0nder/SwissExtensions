# Video Downloader

Расширение открывает один загрузчик (SaveFrom) и передаёт ему вашу ссылку. **Тип ссылки определяет сам сервис** (LinkedIn, Instagram, Twitter и др.) — в расширении нет логики «платформа → сервис».

## Как пользоваться

1. Скопируйте ссылку на пост/видео (любая поддержанная SaveFrom: LinkedIn, Instagram, Twitter и т.д.).
2. Нажмите на иконку расширения, вставьте ссылку в поле.
3. Нажмите «Открыть загрузчик» — откроется вкладка SaveFrom с переданной ссылкой; сервис сам определит источник и предложит скачать.

## Сервис

Используется [SaveFrom](https://ru.savefrom.net/246rR/) — один URL для всех поддерживаемых сайтов. Позже можно добавить YouTube (видео и музыка).

## Установка

Chrome: `chrome://extensions` → «Загрузить распакованное расширение» → папка `VideoDownloader`.
