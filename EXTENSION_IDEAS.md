# Идеи расширений — что нужно юзерам

## Продуктивность и фокус
- **Таймер Pomodoro** — блок пауз, фиксированная длительность сессий
- **Read-it-later** — сохранить статью на потом (как Pocket)
- **Session / вкладки** — сохранить набор вкладок, открыть одной кнопкой
- **Трекер времени на сайтах** — сколько провёл на YouTube, соцсетях
- **Лимит времени на сайтах** — не только блокировка, но и «макс N минут в день»

## Приватность
- **Очистка cookies при закрытии** — автоматически или по сайтам
- **Блок трекеров** — uBlock-подобное
- **Удаление fingerprint** — fake user-agent, canvas и т.д.
- **Автоудаление истории** — через N минут / при закрытии вкладки

## UX и удобство
- **Тёмная тема везде** — Dark Reader
- **Увеличение шрифта** — глобально или по сайтам
- **TL;DR статей** — краткое содержание
- **Перевод по выделению** — hover-перевод
- **Восстановление закрытых вкладок** — список + поиск
- **Сохранение закрытых вкладок** — умнее чем Cmd+Shift+T

## Контент
- **Adblock** — блок рекламы
- **Video speed controller** — 0.5x–4x на любом видео
- **Скачивание видео/аудио** — с YouTube и др.
- **Скрыть Shorts/Reels** — убрать бесконечную ленту
- **Убрать рекомендации YouTube** — только подписки

## Для разработки
- **JSON/XML viewer** — читаемый вывод
- **Clear cache for domain** — без перезагрузки
- **Copy as cURL** — из Network
- **API tester** — быстрый REST-клиент

## Популярные запросы в поиске
ad blocker, dark mode, translate, youtube download, password manager, vpn, tab manager, session saver, screenshot, pdf converter

---

## Топ-3 (из популярных запросов)

1. **Dark mode** — тёмная тема для любого сайта. Высокий спрос, понятная задача.

2. **Translate** — перевод по выделению / при наведении. Востребовано, без тяжёлой backend-логики.

3. **Session saver** — сохранение и восстановление сессий вкладок. Отличается от Tab Hibernate: явное «Save session» и «Restore session».
